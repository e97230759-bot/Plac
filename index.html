Чтобы добавить сайт на экран телефона как иконку приложения:

Android (Chrome):

  1. Открой сайт в Chrome
  2. Нажми ⋮ (три точки) вверху справа
  3. Выбери "Добавить на главный экран"
  4. Подтверди — появится иконка

iPhone (Safari):

  1. Открой сайт в Safari
  2. Нажми кнопку Поделиться (квадрат со стрелкой вверх) внизу
  3. Выбери "На экран «Домой»"
  4. Подтверди — появится иконка

После этого приложение открывается как обычное — без адресной строки браузера.

Когда нажимаешь "Анализuj dane":

  1. Приложение собирает все текущие данные — конtenery, worki, strefy, wózki, butle, pracownicy, notatki, baza wiedzy
  2. Формирует один большой JSON с этими данными
  3. Отправляет запрос напрямую на api.anthropic.com/v1/messages (Claude Sonnet)
  4. Claude анализирует данные и возвращает текст на польском — факты, рекомендации, тенденции

5:29 AM

Есть три варианта:

1. Ключ прямо в коде (самый простой)

  • Вставляешь sk-ant-... прямо в index.html
  • Работает сразу
  • Минус: кто угодно может открыть код страницы и увидеть ключ — но если приложение только для своих (пароль plac2026), это приемлемо

2. Ключ вводится вручную при входе (безопасно + просто)

  • На экране входа два поля: пароль + API ключ
  • Ключ хранится в sessionStorage — не виден в коде
  • Никто посторонний не получит ключ

3. Proxy-сервер (максимально безопасно, но сложно)

  • Отдельный сервер хранит ключ
  • Приложение обращается к серверу, а не к Anthropic напрямую
  • Требует Netlify Functions или другой бэкенд

Учитывая что приложение закрыто паролем и используется только вами — вариант 1 или 2 вполне достаточно. Вариант 2 немного элегантнее.

ШАГ 1 — Получить ключ Gemini

  1. Открой aistudio.google.com
  2. Войди через Google аккаунт
  3. Нажми "Get API key" → "Create API key"
  4. Скопируй ключ — выглядит как AIzaSy...

ШАГ 2 — Вставить ключ в файл

  1. Открой index.html в любом текстовом редакторе (Notepad, VS Code и т.д.)
  2. Нажми Ctrl+F (поиск)
  3. Найди текст: WSTAW_TUTAJ_KLUCZ_GEMINI
  4. Замени его на свой ключ, например:
const GEMINI_API_KEY = "AIzaSyXXXXXXXXXXXXXXX";
  1. Сохрани файл

ШАГ 3 — Загрузить на Netlify

  1. Открой netlify.com → твой сайт
  2. Вкладка Deploys
  3. Перетащи index.html в зону загрузки или нажми Browse to upload
  4. Подожди 10–20 секунд

ШАГ 4 — Проверить

  1. Открой сайт, войди с паролем plac2026
  2. Перейди в 🤖 Agent AI
  3. Нажми "Analizuj dane"
  4. Через 3–5 секунд появится отчёт в трёх секциях
